Creature-Kitchen-Randomizer-Mod
- Install Instructions -
1.) Install BepInEX Bleeding Edge IL2CPP from here -> https://builds.bepinex.dev/projects/bepinex_be
2.) Place all contents of BepInEx into the root folder for Creature Kitchen
3.) Run game to initialize BepInEx, then close it.
4.) Inside the BepInEx folder there should now be a "plugins" folder. This is where you will place "CreatureKitchenAP.dll"
5.) Run the game again to create config files for the mod, close it after you reach the main menu.
6.) In BepInEx/config folder there is a new file for enabling/disabling features of the mod edit as you please and have fun!

Please report any bugs you find and PLEASE send me a copy of your logoutput.log file so I can see what went wrong!
